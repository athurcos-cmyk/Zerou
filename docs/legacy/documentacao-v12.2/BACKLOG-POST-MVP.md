# Zerou — Backlog pós-MVP

Este backlog registra itens deliberadamente retirados do lançamento inicial. Não antecipar durante as seis fases do MVP salvo quando um scaffold mínimo for indispensável para evitar retrabalho estrutural.

# 1. Organização financeira avançada

- orçamentos completos;
- metas financeiras;
- provisões com ledger;
- dívidas, empréstimos e financiamentos;
- patrimônio líquido;
- fechamento mensal completo;
- snapshots mensais;
- previsão de saldo avançada;
- disponível livre com reservas mínimas e aportes obrigatórios.

# 2. Automação e dados

- inbox avançada;
- regras automáticas;
- detecção de duplicidade aprimorada;
- importação CSV;
- importação OFX;
- importação XLSX;
- exportação XLSX;
- relatórios avançados;
- OCR por adapter com confirmação obrigatória;
- anexos avançados;
- conciliação avançada.

# 3. Experiência visual

Os seis temas oficiais pertencem à Fase 1. Este backlog mantém somente refinamentos adicionais posteriores ao lançamento.


- landing 3D com fallback;
- Three.js;
- `@react-three/fiber`;
- `@react-three/drei`;
- refinamento visual adicional após métricas reais de uso.

# 4. Administração e operação

- painel admin completo;
- reprocessamento manual de webhook por UI;
- tickets internos;
- métricas operacionais avançadas;
- auditoria administrativa detalhada;
- feature flags administráveis;
- painel de custos;
- observabilidade aprofundada.

# 5. Notificações

- FCM;
- push;
- preferências granulares;
- lembretes avançados;
- campanhas transacionais sofisticadas.

# 6. Pagamentos adicionais

- Pix Automático somente após validação de disponibilidade na conta Stripe e desenho comercial;
- boleto quando compatível;
- cupons e promoções avançadas.
