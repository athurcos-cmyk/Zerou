# Zerou v12.2 — Pacote completo

Este pacote reúne a documentação incremental do produto e os assets visuais aprovados da marca Zerou.

## Comece por aqui

1. Leia `documentacao-v12.2/README.md`.
2. Leia `documentacao-v12.2/HANDOFF-CHECKLIST.md` e `documentacao-v12.2/THEME-SYSTEM.md`.
3. Execute `documentacao-v12.2/00-BOOTSTRAP-MANUAL.md` até onde for possível.
4. Copie os arquivos necessários de `assets-visuais/` conforme `documentacao-v12.2/BRAND-ASSET-INTEGRATION.md`.
5. Entregue ao agente apenas um prompt operacional por vez.

## Marca oficial

```text
Zerou
Controle individual. Organização a dois.
```

## Observação

Não executar as seis fases em uma única solicitação. Cada prompt termina com gate e parada obrigatória.

## Temas

A interface autenticada possui seis temas configuráveis por usuário desde a Fase 1. A identidade da marca permanece estável; as cores dos componentes são resolvidas por tokens semânticos.
